Scapy: build packets like Lego

Stack protocol layers with the / operator, send them, capture replies. The most powerful four-line scripts you'll write all year.

27 Jan 2026 • ~3 min read • Python 3.7+
TL;DR

Scapy treats every protocol layer as a Python object. Stack them with /: IP(dst="x") / ICMP(). Send with sr1 (one reply), sr (many), or send (no reply). Capture with sniff. Needs root or CAP_NET_RAW.

For anything below the application layer — ARP scans, custom traceroutes, packet replay, protocol fuzzing — nothing else in Python comes close. Scapy is what tcpdump, nmap, and hping would look like if they were a Python library instead of separate binaries.

The stacking syntax

pythonfrom scapy.all import IP, ICMP, TCP, UDP, Ether, ARP, sr1, srp

# Each layer is a class. Stack with /
pkt = IP(dst="8.8.8.8") / ICMP()

# Inspect
pkt.summary()         # 'IP / ICMP 0.0.0.0 > 8.8.8.8 echo-request 0 / Raw'
pkt.show()            # pretty per-field dump
bytes(pkt)            # wire-format bytes

One-line ping

python# Send one packet, wait for one reply, 2-second timeout
reply = sr1(IP(dst="8.8.8.8") / ICMP(), timeout=2, verbose=False)

if reply:
    print(f"alive, ttl={reply.ttl}, rtt={(reply.time - reply.sent_time) * 1000:.1f}ms")
else:
    print("no reply")

The three send functions to remember: sr1 (send and receive one), sr (send and receive many, returns answered/unanswered), send (fire and forget). Add p to any of them (srp1, srp, sendp) to operate at layer 2 instead of layer 3.

An ARP scan in three lines

pythonans, _ = srp(
    Ether(dst="ff:ff:ff:ff:ff:ff") / ARP(pdst="192.168.1.0/24"),
    timeout=2, verbose=False,
)
for _, r in ans:
    print(f"{r.psrc:<15} {r.hwsrc}")

That's a fully functional ARP-based host discovery for a /24. srp understands that pdst is a CIDR and fans out for you.

A TCP-SYN port probe

pythondef syn_probe(host, port):
    r = sr1(IP(dst=host) / TCP(dport=port, flags="S"),
            timeout=1, verbose=False)
    if r is None:                       return "filtered"
    if r.haslayer(TCP) and r[TCP].flags == 0x12:  return "open"
    if r.haslayer(TCP) and r[TCP].flags == 0x14:  return "closed"
    return "unknown"

The flag values: 0x12 is SYN+ACK (open), 0x14 is RST+ACK (closed). Reading TCP flags as hex feels wrong the first time and natural after the second.

Sniffing live traffic

pythonfrom scapy.all import sniff

def show(pkt):
    if pkt.haslayer("DNS") and pkt["DNS"].qr == 0:
        print(pkt["DNS"].qd.qname.decode())

# BPF filter, like tcpdump
sniff(filter="udp port 53", prn=show, store=False, count=20)

store=False matters — without it Scapy keeps every packet in memory. prn is your callback per packet. The filter is a real BPF expression, evaluated by the kernel before the packet ever reaches Python.

Scapy needs raw socket access — that means root, or setcap cap_net_raw,cap_net_admin=eip on your Python binary. On a shared system, prefer the capability to running as root.

Write a 10-line traceroute. Loop TTL from 1 to 30, send IP(dst=target, ttl=N) / ICMP(), capture the reply, print the source IP. The ICMP "time exceeded" replies trace your path one hop at a time. You've just rebuilt traceroute.
